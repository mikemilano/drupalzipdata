drupalzipdata by mikemilano

Drupal 7 module which provides a way to import data from zip-codes.com and query/cache range queries between zip codes.

Drupal 7 module which provides a way to import data from zip-codes.com and query/cache range queries between zip codes. I'm putting it here instead of on drupal.org because there are multiple feeds from that service that I do not pay for and I have no way of supporting or testing them. This would be a good starting point for another developer looking to integrate zipcode range based searches in Drupal.
